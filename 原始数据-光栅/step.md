# 光栅原始图像分析步骤

更新时间：2026-06-25

## 当前进度

已确认本目录包含五个盐水浸泡浓度组、三次重复拍摄、两种图像形态。

当前阶段目标不是直接训练模型，而是先判断光栅图像是否具备可重复、可标定、可解释的信号。

## 待完成工作

1. 建立样本索引表。
2. 检查图像质量与命名一致性。
3. 裁剪或确认光谱 ROI。
4. 提取一维伪光谱曲线。
5. 评估重复稳定性和浓度梯度趋势。
6. 决定是否进入 PLSR。

## 执行步骤

### 1. 样本索引

从文件名解析以下字段：

| 字段 | 说明 |
|---|---|
| `file_path` | 原始图片路径 |
| `concentration_mg_l` | 浸泡盐水浓度，清水组为 0 |
| `group_label` | 对照、2.0、4.0、6.0、8.0 |
| `repeat_id` | 0、1、2 |
| `image_type` | `raw` 或 `focused` |
| `source_mode` | `real_experiment_raw_image` |

### 2. 图像质量检查

每张图至少计算：

- 图像尺寸；
- 平均亮度；
- RGB 通道均值；
- 过曝比例；
- 欠曝比例；
- 清晰度指标；
- 是否存在明显裁剪失败、遮挡、强反光或异常文件大小。

异常必须先记录，不要直接丢弃。

### 3. 光谱 ROI

优先目标：

- 只截取光栅条纹区域；
- 避免混入黑框、文字、水果主体、桌面背景；
- 同一批图像使用同一 ROI 规则；
- 如果固定 ROI 不稳定，再考虑手动确认或半自动检测。

ROI 失败时停止进入建模。

### 4. 伪光谱曲线提取

从 ROI 中提取：

- 灰度平均强度曲线；
- R/G/B 三通道平均曲线；
- 归一化曲线；
- 可选：一阶差分、峰位、峰宽、积分强度、色带中心位置。

注意：

- 当前图像没有真实波长标定；
- 横轴或纵轴只能称为像素位置轴；
- 未标定前不要称为真实光谱波长。

### 5. 重复稳定性判断

先分别评估：

- `raw` 图像；
- `focused` 图像。

判断标准：

- 同一浓度组内三次重复曲线是否相近；
- 不同浓度组之间差异是否大于组内重复差异；
- 曲线是否随浓度出现单调或近似单调变化；
- 是否存在某一重复编号整体偏离，提示拍摄条件改变。

### 6. PLSR 进入条件

只有满足以下条件，才进入 PLSR：

- ROI 稳定；
- 组内重复可接受；
- 组间差异大于组内噪声；
- `raw` 或 `focused` 至少有一种图像形态表现出可分趋势；
- 交叉验证可以按重复编号或样本组分组，避免 `raw/focused` 泄漏。

### 7. PLSR 验证规则

禁止：

- 随机按图片切分训练/测试；
- 把同一重复样本的 `raw` 与 `focused` 分到训练和测试两边；
- 用训练集拟合效果宣称模型有效；
- 用很高的 R2 直接宣称真实盐分检测成功。

推荐：

- 分开建模：只用 `raw`，只用 `focused`；
- 或把同一重复的双视图作为同一样本特征融合；
- 采用 Leave-One-Repeat-Out 或按重复组留出；
- 输出 RMSE、MAE、R2 以及预测-真实浓度图；
- 同时报告样本量不足带来的不确定性。

## 验证标准

光栅路线可继续的最低标准：

- 光谱条纹清晰且 ROI 可稳定裁剪；
- 同浓度重复之间差异较小；
- 不同浓度之间存在稳定可分趋势；
- 趋势不是由过曝、聚焦、背景、果皮位置变化造成；
- PLSR 留组验证结果明显优于均值预测基线。

## 验收条件

阶段一验收：

- 生成样本索引表；
- 生成图像质量统计；
- 生成 ROI 可视化检查图；
- 生成各组伪光谱曲线图；
- 给出 `raw` 与 `focused` 哪个更适合后续分析的证据；
- 明确结论：进入 PLSR、暂缓 PLSR、或放弃光栅路线改走受控 RGB 成像。

## 输出要求

后续分析输出必须区分：

- Facts：文件、图片、命令、统计结果直接支持的事实；
- Inferences：基于事实的推断；
- Unknowns：尚未证明或仍缺失的信息；
- Allowed Conclusion：当前证据允许的结论；
- Forbidden Conclusion：当前证据不允许的结论。
