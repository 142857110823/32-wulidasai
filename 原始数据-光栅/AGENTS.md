默认使用中文。

本目录只服务于 FreshSalt Surface / 表面盐影像助手 的光栅反射原始图像分析。

## 2. 数据身份

本目录数据是光栅反射实验的原始图片，用于判断不同盐水浸泡浓度处理后，圣女果反射光透过光栅形成的图像是否存在可重复的伪光谱差异。

数据不是：

- 食品安全检测数据；
- 执法检测数据；
- 已完成标定的真实表面盐分含量数据；
- 可直接宣称测盐成功的数据。

## 3. 默认科学边界

必须坚持以下边界：

- 标签是浸泡盐水浓度，不是水果表面真实盐分含量；
- `mg/L` 单位在未经用户更正前不得擅自改成 `g/L`；
- `raw` 与 `focused` 是同一样本的两种图像形态，不是两个独立样本；
- PLSR 只能在通过 ROI 和重复稳定性门禁后执行；
- 未做波长标定前，光谱横轴只能称为像素位置轴；
- 未做真实盐分标定前，结论只能写“浓度组响应”或“光学响应”，不能写“真实盐分含量预测”。

## 4. 文件命名规则

文件名前缀：

| 前缀 | 含义 |
|---|---|
| 无前缀 | 第 0 次重复 |
| `-1` | 第 1 次重复 |
| `-2` | 第 2 次重复 |

文件后缀：

| 后缀 | 含义 |
|---|---|
| `-0` | 原生图片 |
| `-1` | 聚焦图片 |

浓度组：

| 文件文字 | 标签 |
|---|---:|
| `对照` | 0 mg/L |
| `盐浓度2.0mgL` | 2.0 mg/L |
| `盐浓度4.0mgL` | 4.0 mg/L |
| `盐浓度6.0mgL` | 6.0 mg/L |
| `盐浓度8.0mgL` | 8.0 mg/L |

## 5. 默认分析协议

后续分析必须按以下顺序：

1. 文件索引和命名核对；
2. 图像质量统计；
3. ROI 可行性检查；
4. 伪光谱曲线提取；
5. 重复稳定性分析；
6. 浓度梯度趋势分析；
7. 决定是否进入 PLSR；
8. 如果进入 PLSR，必须使用防泄漏验证；
9. 输出结论边界和下一步建议。

不得跳过 1-6 直接训练模型。

## 6. PLSR 默认规则

PLSR 是候选方法，不是默认结论。

允许：

- 分别评估 `raw` 和 `focused`；
- 对同一样本的 `raw/focused` 做特征融合；
- 使用分组交叉验证；
- 与均值预测基线比较。

禁止：

- 按单张图片随机划分训练集和测试集；
- 把同一重复样本的 `raw` 与 `focused` 分入不同数据集；
- 用训练集拟合效果证明模型有效；
- 用小样本高 R2 直接宣称检测成功。

## 9. 停止条件

出现以下情况必须先停下并说明：

- 命名规则无法可靠解析；
- `mg/L` 单位与实际实验记录不一致；
- 图片缺失或组别数量不匹配；
- ROI 无法稳定裁剪；
- 同组重复差异大于组间差异；
- 用户要求直接宣称真实盐分含量；
- 需要删除、覆盖或移动大量原始图片。

## 10. 当前推荐路线

首选路线：

```text

-> 图像质量门禁
-> 光谱 ROI
-> 一维伪光谱曲线
-> 重复稳定性 / 浓度趋势

```
